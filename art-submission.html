Ready to send us your art files? Great!  Heres a few guidelines and suggestions to help make everything go smoothly.

Send files to jsmerdel@cspgraphics.com.  If sending files over 10mb and do not have your own FTP we suggest WeTransfer.com

 

 

 

•  Acceptable file formats:  INDD (InDesign), AI (Illustrator), PSD (Photoshop), PDF (see below for info), EPS, TIFF.  We may be able to accept file formats not listed here, if you have something that is not one of the above formats please contact us to determine the best way to handle the art.

•  Documents should be set to final trim size.  If your finished size is 12" x 12", then your document size should match that.   We prefer you not supply art that requires scaling, cropping or resizing of art.

•  Bleed should be set to .25" (1/4") and if in PDF format should NOT contain trim marks or any other "printers marks".

•  If your art contains type/copy it should either be outlined or fonts should be supplied with art.

•  If your art contains a dieline, the stroke should be set to spot color and titled as "DIE".

•  If your art contains raster images keep them at the highest quality possible.

•  If art contains links to images those images should be supplied with your art.

•  If you require a specific Pantone color make sure that color is specified in the art.   Objects that you intend to print as spot color should be set to that color in the document.
